# Day 29 — Interview Q&A & Failure Scenarios

## Failure Handling
- Bedrock throttling → retries & fallback
- OpenSearch failure → snapshots & restore
- Cost spikes → per-tenant throttling

## Trade-Offs
- Bedrock vs self-hosted LLM
- OpenSearch vs external vector DB
- IAM auth vs JWT

## Interview Strategy
- Focus on resilience, cost, and security
- Explain trade-offs clearly
