# Enterprise GenAI Knowledge Assistant (AWS)

## Overview
A secure, multi-tenant, production-grade GenAI knowledge assistant built on AWS using Bedrock and Retrieval-Augmented Generation (RAG).

## Key Capabilities
- Tenant-isolated RAG using OpenSearch
- Secure API with IAM authentication
- Versioned documents and embeddings
- Cost tracking and alerts per tenant
- Full audit logging and observability
- Disaster recovery and compliance-ready design
